---
- name: Install Nginx on Pathnex server
  hosts: all
  become: yes

  tasks:
    - name: Install nginx
      yum:
        name: nginx
        state: present
====================================

Explanation
--- → Start of YAML file

name → Playbook description

hosts: all → Run on all servers

become: yes → Use sudo/root privileges

tasks → List of tasks to execute

yum → Package manager module for RHEL/CentOS/Amazon Linux

name: nginx → Package to install

state: present → Ensure package is installed

=============================================
0 spaces   → Main section
2 spaces   → Task
4 spaces   → Module
6 spaces   → Module values
